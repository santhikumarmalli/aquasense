# AquaSense API Documentation

## Authentication API

### Register User
```
POST /api/auth/v1/register
```

### Login
```
POST /api/auth/v1/login
```

## Sensor API

### Get Sensor Data
```
GET /api/sensors/v1/data
```

### Submit Reading
```
POST /api/sensors/v1/readings
```

For complete API documentation, visit: http://localhost:8080/swagger-ui.html
